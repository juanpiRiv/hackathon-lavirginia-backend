# Dataset de trabajo

## Regla base
- `1 foto = 1 caja/unidad`.
- No mezclar varias cajas en la misma imagen.
- Mantener la unidad centrada y visible.

## Carpetas sugeridas

```text
dataset/
  raw/
  curated/
  templates/
```

## Checklist de captura
- fondo simple y mate
- luz difusa y estable
- una sola unidad por foto
- sin blur
- sin reflejos que tapen la zona relevante
- mantener angulo y distancia consistentes

## Metadata
Usar `templates/metadata_template.csv` como fuente de verdad.
